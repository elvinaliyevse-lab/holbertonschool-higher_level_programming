# python-if_else_loops_functions

## Description
This directory contains small Python scripts that demonstrate basic control structures, loops, and functions using simple arithmetic and string operations.

## Files and explanations

- `0-positive_or_negative.py`: Determines whether a randomly generated integer between -10 and 10 is positive, negative, or zero.
- `1-last_digit.py`: Prints the last digit of an integer using arithmetic operations.
- `2-print_alphabet.py`: Prints the lowercase alphabet followed by a new line.
- `3-print_alphabt.py`: Prints the lowercase alphabet except for `q` and `e`, without a trailing newline.
- `4-print_hexa.py`: Prints all numbers from 0 to 98 in decimal and hexadecimal format.
- `5-print_comb2.py`: Prints all possible combinations of two two-digit numbers from 00 to 99, separated by a comma and space.
- `6-print_comb3.py`: Prints all combinations of two digits, separated by a comma and space, without repeating the same pair twice.
- `7-islower.py`: Defines a function that checks whether a single character is a lowercase letter.
- `8-uppercase.py`: Defines a function that prints a string in uppercase using character code arithmetic.
- `9-print_last_digit.py`: Defines a function that prints and returns the last digit of an integer.
- `10-add.py`: Defines a function that returns the sum of two integers.
- `11-pow.py`: Defines a function that computes the power of a number using exponentiation.
- `12-fizzbuzz.py`: Prints numbers from 1 to 100, replacing multiples of three with `Fizz`, multiples of five with `Buzz`, and multiples of both with `FizzBuzz`.
- `100-print_tebahpla.py`: Prints the lowercase ASCII alphabet in reverse order, alternating lowercase and uppercase letters, without a trailing newline.
- `101-remove_char_at.py`: Defines a function that returns a copy of a string with the character at a given index removed.

## Requirements
- Python 3

## Usage

Run any script with:

```bash
python3 <script_name>.py
```
